It's a collection of 50+ pre-built business skills, 19 specialist AI agents, voice personalities, and mobile bridges — all packaged to install into Claude Code with one command. Built by Daniel Wise for entrepreneurs who'd rather get results than configure software.

Is this a subscription?

No. One-time payment per tier. Lifetime access. No monthly traps.

Do I need Claude Code?

Yes. Claude Code is Anthropic's free CLI tool — download it from anthropic.com. LindaAI runs inside Claude Code.

What platforms are supported?

macOS, Windows, and Linux. Cross-platform installers included in every tier.
