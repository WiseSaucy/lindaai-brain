# LindaAI Agent Avatars — Generation Guide

18 custom avatars, one per agent. These prompts are pre-engineered to produce a **consistent LindaAI look** across the whole roster — same style, same palette, same vibe — so they feel like a family.

## Where to run them

Any one of these works great. Pick ONE and run all 18 prompts through the same tool so the style stays locked:

- **Midjourney** (best quality) — paste the prompt, add `--ar 1:1 --v 6` at the end
- **Ideogram** (ideogram.ai — free tier) — paste as-is
- **ChatGPT Image** (DALL-E 3) — paste as-is
- **Google Gemini / Imagen** — paste as-is
- **Leonardo AI** — use "PhotoReal v2" model

## Output specs
- **Size:** 512×512 (or 1024×1024 and resize)
- **Format:** PNG with transparent background if your tool supports it; otherwise solid LindaAI black
- **Filename:** `<agentname>.png` — lowercase, no spaces (e.g. `bandit.png`)
- **Save to:** `LindaAI-Master/agents/avatars/`

## LindaAI House Style (locked — do not deviate)

Every prompt below starts with this style block. This is what keeps them looking like one family:

> *"Stylized modern country-western portrait, bold confident character, LindaAI brand style: matte black background, warm amber and deep red accents, clean vector illustration with subtle grain, strong rim lighting, slight 3/4 angle, serious but warm expression, centered bust framing. Professional tech-brand mascot energy meets Texas grit."*
